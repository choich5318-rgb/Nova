# New Star - 새로운 시작

신제품 및 신기술 홍보용 단일 페이지 웹 애플리케이션

## 📋 프로젝트 개요

"New Star - 새로운 시작"은 신제품과 신기술을 소개하는 홍보용 단일 페이지 웹사이트입니다. 모든 기능이 하나의 HTML 파일에 포함되어 있어 배포가 간편하며, 모던한 디자인과 부드러운 사용자 경험을 제공합니다.

## ✨ 주요 기능

### 1. 네비게이션 바
- 고정형 상단 네비게이션
- 반투명 배경 및 블러 효과
- 부드러운 스크롤 링크

### 2. 히어로 섹션
- 전체 화면 높이의 인상적인 첫 화면
- 기업 이미지 배경 (50% 투명도)
- 그라데이션 오버레이
- 애니메이션 효과가 적용된 텍스트
- CTA(행동 유도) 버튼

### 3. 주요 특징 섹션
- 6개의 특징 카드
  - 🚀 혁신적인 성능
  - 💡 스마트한 설계
  - 🔒 안전한 보안
  - ⚡ 빠른 속도
  - 🌐 글로벌 지원
  - 🎯 정확한 분석
- 호버 효과 및 카드 애니메이션
- 반응형 그리드 레이아웃

### 4. 신기술 소개 섹션
- 2단 레이아웃
- 시각적 요소와 텍스트 콘텐츠
- 그라데이션 배경

### 5. 통계 섹션
- 4개의 주요 통계 지표
- 시각적으로 강조된 숫자

### 6. 문의 섹션
- 연락처 폼
- 입력 유효성 검사
- 제출 처리 기능

## 🛠️ 기술 스택

- **HTML5**: 시맨틱 마크업
- **CSS3**: 
  - Flexbox 및 Grid 레이아웃
  - CSS 그라데이션
  - 애니메이션 및 트랜지션
  - 반응형 디자인 (미디어 쿼리)
- **JavaScript (Vanilla)**:
  - 부드러운 스크롤 기능
  - 폼 제출 처리
  - Intersection Observer API를 활용한 스크롤 애니메이션

## 🚀 사용 방법

### 방법 1: 직접 열기
1. `index.html` 파일을 웹 브라우저에서 직접 열기

### 방법 2: 로컬 웹 서버 실행
```bash
# Python 3
python -m http.server 8000

# Node.js (http-server)
npx http-server

# PHP
php -S localhost:8000
```

브라우저에서 `http://localhost:8000` 접속
